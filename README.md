# booking_scraper

## Overview

A Puppeteer-based web scraper for extracting hotel information from Booking.com. This project demonstrates automated browser control and data extraction.

## Features

- Real-time data extraction (prices, ratings, urls)
- Headless browser automation
- Error handling
- Results export to JSON/CSV

## Prerequisites

- Node.js 14+
- npm or yarn

## Installation

```bash
npm install
```




## Project Structure

```
├── src/
│   ├── index.js      # Main scraping logic
│   └── config.js       # Configuration
├── output/             # Results directory
└── package.json
```
